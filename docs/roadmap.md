# Roadmap de Aprendizado

O roadmap abaixo organiza os estudos por prioridade, mantendo uma evolucao progressiva.

## Fase 1 - Fundamentos

- Logica de programacao
- Git e GitHub
- HTML
- CSS
- JavaScript basico

## Fase 2 - Pratica com projetos

- Manipulacao do DOM
- Formularios
- Eventos
- Validacoes
- Projetos pequenos com JavaScript

## Fase 3 - Front-end moderno

- JavaScript moderno
- React
- Componentizacao
- Rotas
- Consumo de APIs

## Fase 4 - Back-end e dados

- Node.js
- APIs REST
- SQL
- Integracao com banco de dados

## Fase 5 - Portfolio

- Projetos documentados
- README por projeto
- Deploys
- Organizacao de perfil no GitHub
- Preparacao para primeira oportunidade
