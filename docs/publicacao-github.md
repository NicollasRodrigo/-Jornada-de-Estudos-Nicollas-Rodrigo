# Publicacao no GitHub

Este workspace ja e um repositorio Git local. O remote antigo de JavaScript foi preservado como `javascript-origin`:

```text
https://github.com/NicollasRodrigo/CURSO-DE-JAVASCRIPT.git
```

O remote principal `origin` aponta para o hub:

```text
https://github.com/NicollasRodrigo/hub-estudos-programacao.git
```

## Passos finais

1. Confira os arquivos alterados:

```bash
git status
```

2. Faça o commit da reorganizacao:

```bash
git add -A
git commit -m "Organiza hub principal de estudos"
```

3. Envie para o GitHub:

```bash
git push -u origin main
```
