## Padrao recomendado para novos repositorios

- Nome curto, em minusculas e separado por hifen.
- README com objetivo, conteudos, estrutura e projetos.
- Pastas separadas para aulas, exercicios, projetos, documentacao e assets.
- Commits pequenos e descritivos.
